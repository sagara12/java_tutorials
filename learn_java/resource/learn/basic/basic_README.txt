프로그램밍이란?
 하나 이상의 관련된 추상 알고리즘을 특정한 프로그래밍 언어를 이용해
 구체적인 컴퓨터 프로그램으로 구현하는 기술
    https://ko.wikipedia.org/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D

 주어진 문제를 해결하기 위해 프로그래밍 언어를 사용하여 문제 해결 방법을 기술

 문제 해결 방법: 알고리즘

 문제해결방법 해부
   - 정보처리 기법
     . 정보: 숫자 / 문자
     . 전체 문제를 알고리즘으로 정리할 수 있는 작은 단위로 쪼갬 --> 함수(메소드)

   - 정보 저장
   - 정보 전달
   - 정보 계산

 문제를 해결하는 주체: 컴퓨터
   - 컴퓨터 장치의 특성
     . CPU / Controllers 가 할수 있는 것?
       ^ 연산
       ^ 논리적 판단
       ^ 메모리 저장/로딩
       ^ 실행코드 수행
